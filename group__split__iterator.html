This method initializes a SplitIterator structure, storing a reference to the specified String and delimiter, and setting the SplitIterator's position to 0.

Parameters
iteratorA pointer to the SplitIterator to be initialized.
stringA parent to the parent String, which will be split into tokens separated by the specified delimiter
delimiterA String containing a sequence of elements which will be used to separate string into tokens.

◆ split_iterator__next()

bool split_iterator__next ( SplitIterator *  iterator,
String *  ref_token 
)

This method retrieves the next token found in the SplitIterator's string field, and stores it in the provided String.

Parameters
iteratorA pointer to the SplitIterator used to retrieve the next token.
out_tokenAn out parameter. Upon successful return, this will represent the next token contained by the parent String.
Returns
1 if this operation completed successfully - that is, the SplitIterator's parent String contains a next token, and that token is represented by out_token.
0 if the operation did not complete successfully.
