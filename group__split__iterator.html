const Slice *  slice,
const Slice *  delimiter 
)

This method initializes a SplitIterator structure, storing a reference to the specified Slice and delimiter, and setting the SplitIterator's position to 0.

Parameters
iteratorA pointer to the SplitIterator to be initialized.
sliceA parent to the parent Slice, which will be split into tokens separated by the specified delimiter
delimiterA Slice containing a sequence of elements which will be used to separate slice into tokens.

◆ split_iterator__next()

char split_iterator__next ( SplitIterator *  iterator,
Slice *  out_token 
)

This method retrieves the next token found in the SplitIterator's slice field, and stores it in the provided Slice.

Parameters
iteratorA pointer to the SplitIterator used to retrieve the next token.
out_tokenAn out parameter. Upon successful return, this will represent the next token contained by the parent Slice.
Returns
1 if this operation completed successfully - that is, the SplitIterator's parent Slice contains a next token, and that token is represented by out_token.
0 if the operation did not complete successfully.
