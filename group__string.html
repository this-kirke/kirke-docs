void string__initialize__format ( String *  string,
Allocator *  allocator,
const char *  format,
  ... 
)

This method initializes a String using the provided format and specifiers. The provided Allocator is used to allocate new memory.

Parameters
stringA pointer to the String which will be initialized.
allocatorThe allocator which will be used to manage memory controlled by the provided String.
formatA null-terminated (C-style) string of characters, with optional format specifiers. Format specifiers are the same as those used for printf. If format specifiers are contained within format, then this parameter must be followed by a variable argument list of the corresponding variables.

argsIf format specifiers are contained within format, then this parameter must contain the arguments which correspond to each specifier used.

◆ string__literal()

String string__literal ( const char *  text)

This method creates a String object containing the specified text.

Parameters
textA pointer to the an array of characters which will be contained by the returned String. This array must be null-terminated.
Note
If a stack-allocated string is passed as text, then the data contained within the returned string must not be modified. It is the user's responsibility to ensure this is followed; no string__ methods which modify the data should be called.
