A convenience method which creates a new Error structure, dynamically-allocated. The parameters type and format are not owned; their memory must be mananged by the caller. The purpose of this method is to allow Error creation with string literals and format strings.

Parameters
allocatorA pointer to the Allocator which will be used to allocate memory for the new Error structure.
typeA null-terminated C-style string which describes the general type of the message, such as the class in which the error originated.
codeAn identifier for this particular error, unique within its type.
formatA format specifier with variable arguments for string replacement, which are used for creating the message field
Returns
A pointer to the new Error structure, initialized with the provided type, code and message.

Error* error__new__va_list ( Allocator *  allocator,

va_list  args

A convenience method which creates a new Error structure, dynamically-allocated. The parameters type and format are not owned; their memory must be mananged by the caller. The purpose of this method is to allow Error creation with string literals and a va_list for string replacement.

Parameters
allocatorA pointer to the Allocator which will be used to allocate memory for the new Error structure.
typeA null-terminated C-style string which describes the general type of the message, such as the class in which the error originated.
codeAn identifier for this particular error, unique within its type.
formatA format specifier with string replacement tokens, in the style of printf, which are used for creating the message field.
argsA va_list containing the arguments for string replacement tokens in format.
Returns
A pointer to the new Error structure, initialized with the provided type, code and message.
