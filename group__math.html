◆ math__nearest_greater_power_of_2__ulong()

unsigned long math__nearest_greater_power_of_2__ulong ( unsigned long  value)

This method calculates and returns the next greater power of 2 for a given value.

Parameters
valueThe value for which the next greater power of 2 will be calculated.
Note
If the value itself is a power of 2, this method still returns the next greater power of 2. For example, math__nearest_greater_power_of_2__ulong( 2 ) == 4.
