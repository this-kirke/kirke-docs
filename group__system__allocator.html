◆ system_allocator__deinitialize()

void system_allocator__deinitialize ( SystemAllocator *  system_allocator)

De-initializes a SystemAllocator structure. This must be called when finished with a SystemAllocator initialized with system_allocator__initialize.

Parameters
system_allocatorA pointer to the SystemAllocator structure to be de-initialized.
Note
Internally this calls allocator__destroy on Allocator, so failing to call this method will leak memory.

void system_allocator__initialize ( SystemAllocator *  system_allocator,
void(*)(void)  out_of_memory_callback 
)

Initializes a SystemAllocator structure. The SystemAllocator structure contains an Allocator field which can be passed to any method taking an Allocator* parameter. This allocator is a vtable which points to the system's malloc, realloc and free methods.

Parameters
system_allocatorA pointer to the SystemAllocator to be initialized.
out_of_memory_callbackAn optional method which will be called if alloc or realloc return NULL, i.e. an out of memory condition occurred.
